## Install

- `pip install nisanyan-cli`

## Usege

- `nis <word>`

### Arguments

```
  -h, --help     show this help message and exit
  -p, --plain    returns plain output
  -v, --version  show program's version number and exit
```

# Requirements

- Python >3.5

- [requests](https://pypi.org/project/requests/)
